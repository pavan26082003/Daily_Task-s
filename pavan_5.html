Emmet Abbreviations in VS Code

Emmet is a plugin that lets you write HTML and CSS code more efficiently using abbreviations. Here are some common **Emmet abbreviations**:

HTML Abbreviations:

1. `!`

- Expands to a full HTML5 boilerplate.

2. `ul>li*5`

- Creates a <'ul> with 5 <li >elements. (Use < for < and > for > to ensure the angle brackets are displayed as text, not interpreted as HTML tags.)

3. `div#container>div.item*3`

- Creates a `
` with an `id="container"` containing three `
` elements with the class `item`.

4. `a[href="https://example.com"]`

- Creates an anchor tag with an `href` attribute set to "https://example.com".

5. `table>tr*3>td*2`

- Creates a table with 3 rows and 2 cells in each row.

CSS Abbreviations:

1. `m10`

- Expands to `margin: 10px;`.

2. `p10-20`

- Expands to `padding: 10px 20px;`.

3. `bgc`

- Expands to `background-color`.

4. `w100`

- Expands to `width: 100px;`.

Keyboard Shortcuts in VS Code

Here are some commonly used shortcuts in VS Code to boost productivity:

Basic Shortcuts:

- **Open Command Palette:**

- Windows/Linux: `Ctrl + Shift + P`
- Mac: `Cmd + Shift + P`

- **Quick Open File:**

- Windows/Linux: `Ctrl + P`
- Mac: `Cmd + P`

- **Save File:**

- Windows/Linux: `Ctrl + S`
- Mac: `Cmd + S`

- **Close File:**

- Windows/Linux: `Ctrl + W`
- Mac: `Cmd + W`

- **New File:**

- Windows/Linux: `Ctrl + N`
- Mac: `Cmd + N`

Editing Shortcuts:

- **Copy Line:**

- Windows/Linux: `Alt + Shift + Down/Up Arrow`
- Mac: `Option + Shift + Down/Up Arrow`

- **Move Line Up/Down:**

- Windows/Linux: `Alt + Up/Down Arrow`
- Mac: `Option + Up/Down Arrow`

- **Comment/Uncomment Line:**

- Windows/Linux: `Ctrl + /`
- Mac: `Cmd + /`

- **Select All:**

- Windows/Linux: `Ctrl + A`
- Mac: `Cmd + A`

Navigation Shortcuts:

- **Go to Line:**

- Windows/Linux: `Ctrl + G`
- Mac: `Cmd + G`

- **Go to Definition:**

- Windows/Linux: `F12`
- Mac: `F12`

- **Search Within File:**

- Windows/Linux: `Ctrl + F`
- Mac: `Cmd + F`

Multi-Cursor Shortcuts:

- **Add Cursor at Next Match:**

- Windows/Linux: `Ctrl + D`
- Mac: `Cmd + D`

- **Select Multiple Lines:**

- Windows/Linux: `Alt + Click`
- Mac: `Option + Click`
